这是一个模拟独立钻石（或称为孔明棋）（英文：Peg Solitaire）英式棋盘的计算机游戏，通过键盘输入进行操作，具体玩法如下：
1.将棋盘放在一个笛卡尔坐标系中：

6     X X X
5     X X X
4 X X X X X X X
3 X X X X X X X
2 X X X X X X X
1     X X X
0     X X X
  0 1 2 3 4 5 6

2.输入坐标
3.输入操作（用英文字母表示）如下：
左跳：l
右跳：r
上跳：u
下跳：d
4.按下回车键

例如（[1m粗体[0m为玩家的输入）：
    X X X
    X X X
X X X X X X X
X X X O X X X
X X X X X X X
    X X X
    X X X
[1m13r[0m
    X X X
    X X X
X X X X X X X
X O O X X X X
X X X X X X X
    X X X
    X X X
